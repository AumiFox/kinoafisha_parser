# kinoafisha_parser
Краткое описание проекта:
Парсер для сбора информации об оценках фильмов пользователя КиноАфиша.

Задачи:
При создании рекомендательной системы фильмов необходимо иметь данные о предпочтениях пользователей. КиноАфиша — это популярный сайт о кино, где пользователи оценивают просмотренные фильмы.

                                                                            Цель проекта:
Разработать парсер, который принимает уникальный идентификатор пользователя КиноАфиша и возвращает информацию об оценках фильмов этим пользователем.

                                                                           Результат:
Структурированные данные (название фильма, оценка пользователя) в формате Excel.

                                                                            Установка и запуск
 1.Установка зависимостей

```bash
pip install cloudscraper beautifulsoup4 pandas openpyxl

bash
python kinoafisha_parser.py
